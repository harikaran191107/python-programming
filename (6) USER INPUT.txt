USER INPUT 

user input:
It means getting an input from the user at the time of execution of the program and getting the output

syntax:

variable name=datatype(input())
print(variable)

example program:

num1=int(input("enter the number 1:"))
num2=int(input("enter the number 2:"))
sum=num1+num2
print("The sum of two numbers", sum)

output:
enter the number 1: 3
enter the number 2: 4
The sum of two numbers is 7

In this,
num1,num2,sum-->variables
int-->datatype
()-->paranthesis
print-->used to print the output
input-->used to get the user input

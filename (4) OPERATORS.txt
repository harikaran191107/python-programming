OPERATORS:
Operators are symbols used to perform operations on variables and values.

example:
a + b

In this,
"+" is an operator.

Types of Operators: 

i. Arithmetic Operators:
This operator is used for performing mathematical calculations.

"+"-->addition
"-"-->subtraction
"*"--.multiplication
"\"-->division
"%"-->modulo

example:
a=10
b=3
print(a+b)
print(a-b)
print(a*b)
print(a/b)
print(a%b)
print(a**b)
print(a//b)

output:13 7 30 3 1 1000 3

ii. Comparison Operators:
This operator is used to perform the comparison of values.

< -->greater than
> -->lesser than
<= -->greater than and equal
>= -->lesser than and equal
== --> equal
!= -->not equal

example:
a = 10
b = 5
print(a>b)
print(a<b)
print(a>=b)
print(a>=b)
print(a!=b)
print(a==b)

output:0 1 0 0 0 1


iii. Logical Operators:
This operator is used to combine conditions.

"and"-->true if both conditions are true
"or"-->true if at least one condition is true 
"not"-->reverse the condition

example:
a=10
print(a>5 and a<20)
print(a>15 or a<20)
print(not(a>5))

output:1 1 0

iv. Assignment Operators:
This operator is used to assign values.

"="-->equals to
"+="-->addition
"-="-->subtraction
"*="--.multiplication
"\="-->division

example:
x==10
x+=5
x-=5
x*=5
x/=5
print(x)

output:10 15 5 50 2

vi. Bitwise Operators:
This operator is used to operate on binary bits.
	
"&"-->Bitwise AND(both condition are true)
"|"-->Bitwise OR(any one condition is true)	
"^"-->XOR(true means false)
"~"-->Not

example:
a=5
b=3
print(a&b)
print(a|b)
print(a^b)
print(a~b)

output:1 7 6 -6 

vii. Membership Operators
These operators are used to check whether a value exists in a sequence.

Operator      Meaning
in	      Present
not in	      Not present

example:
name="Python"
print("P" in name)
print("z" not in name)

output:1 1

viii. Identity Operators
These operators are used to compare memory locations.

Operator    Meaning
is	    Same object
is not	    Different object

example:
a=[1,2]
b=a
print(a is b)

output:1
